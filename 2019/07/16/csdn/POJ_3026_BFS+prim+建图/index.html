POJ_3026_BFS+prim+建图

POJ 3026

  • 题意:求出图中连接S和A在图中的最短距离。
  • 样例的意思是S出发步数为0.
  • 难点：
  1. 在读入时用map<pair<int,int>,int>将坐标上的每一个点按顺序从1离散出来.
  2. 通过bfs建图.一开始我以为单点bfs,后来才用多点bfs建图.能使所有点之间边都存在
  3. prim求最短路.
  4. 题目数据读入有空格,也要读(首先就写读入数据)
算法
BFS+prim

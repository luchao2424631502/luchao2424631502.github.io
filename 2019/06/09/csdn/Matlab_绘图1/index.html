Matlab_绘图1

plot(x,y(x));
plot(y(x)); x的长度从1~length(y)

1.同时画多个图,上一个会被refresh,用command
hold on hold off,能在一个窗口上叠加画图像

  • plot(x,y,'str')
  1. str包括三种
  2. Data markers数据标记 (x点的形状)
  3. Line types 连接点之间的线
  4. Colors 线的颜色
    例 plot(x,cos(x),'x--g')

  • 也可以不用hold on来画多个图

  • plot(,,,...)参数多次声明就行了,

  • 标记出来每个函数的名字
    legend('','',...),参数是按照plot函数参数的顺序来计算的

  • 图 X Y Z轴的label 标签,和图的title (title ans ?label)

  • text(x,y,'str') 在(x,y)处添加文本

  • 注意 函数指针 和 函数 作为plot()参数时的写法

    Figure adjustment(图象设置)

    get(gca) %get current axes的属性
    同理get(gcf) %get current figure 的属性

    例如 $sin(x) [0,2\pi]$

  • 由point来修改line的属性

  • 删除对象(指针)释放内存,delete(point)

    设置 线上的点的边

  • figure创建图窗窗口(多个窗口)

    gcf , gca指向最后生成的窗口

  • subplot(n,m,1) %several small plots 'in a figure',几个图像在一个窗口中
    将一个窗口分为n x m个,指定下一个画图区域
    axis style的应用

  • 2种常用存图 对比存数据多了as,即saveas

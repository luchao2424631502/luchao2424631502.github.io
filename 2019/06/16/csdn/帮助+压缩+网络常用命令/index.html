帮助+压缩+网络常用命令

  • man :manual 手册 命令的帮助或者配置文件的帮助
    1. man ls
    2. man services
    3. / 是在文件内搜索
      whereis passwd
      1是命令的帮助
      5是配置文件的帮助
    • 例如:
      whereis passwd 一个是命令,一个是配置文件的名字,

      man passwd默认打开的是命令的帮助文档
      man 5 passwd 打开的是配置文件的文档(帮助信息)
  • whatis ls: 显示命令的名称解释
  • date 日期
  • help 查看shell内置命令
    1. help umask
    2. which umask找不到shell内置命令的绝对路径

用户管理

  • useradd
  • who:查询登录信息
    tty: 本地终端
    pts: 远程终端
  • w:查询详细登录信息
    up time 服务器持续运行时间
    load average服务器负载
    IDLE(懒惰):未下指令持续的时间
    PCUP:cup占用情况
    JCUP:cup持续占用情况

压缩(压缩目录需要先打包在压缩)

  • gzip boduo:GNU zip文件格式(压缩文件).gz

  • gunzip boduo :解压缩

  • tar -cvf [压缩后文件名][目录]
    tar :tape archive(磁带档案)
    -c:建立压缩档案(打包)
    -x:解开包(和打包对应)
    -v:显示所有过程
    -f:(最后一个参数必须加):
    -z:压缩
    1. 先打包在压缩
    tar -cvf Japan.tar Japan
    2.打包同时压缩
    tar -zcvf Japan.tar.zip Japan
    3.打包
    tar -cvf Japan.tar Japan
    4.解包
    tar -xvf Japan.tar

    tar.gz tar.bz2都可以先转成tar,然后在解包

  • zip：保留原文件,文件、目录都可以压缩,所以要先压缩后的文件名
    zip nihao.zip nihao
    zip -r Japan.zip Japan:-r 递归处理文件和子目录

  • unzip nihao.zip

  • bzip2 boduo

    1. 文件格式.bz2
    2. bzip2 -k boduo :-k keep压缩同时保留原文件

  • bunzip2 bodo.bz2

    网络

  • 给其他(在线)用户发送信息

  • wall:write all,给所有用户发信息

    1. wall hello

  • ping 192.168.152.1发送数据包,检测主机是否存在

    1. ping -c 100 192.168.152.1:-c次数

  • ifconfig:interface(接口) configure(配置)

    1. ifconfig eth0 192.168.152.2 临时更改

  • mail root 右面接着内容

    1. mail查看邮件

  • last 登录系统的用户信息

  • lastlog 所有用户最后一次登录的信息

    1. lastlog -u luchao(或者uid),
    2. id luchao:用户的 uid

  • traceroute www.sina.com.cn显示数据包到主机的路径

  • setup配置网络地址(redhat独有)

  • netstat

    1. 端口：接收的服务

      tcp与udp的区别

      只有接收端口固定
    2. netstat -tlun 本机监听的端口
    3. netstat -an 所有ip地址和端口号
    4. netstat -rn 路由表
      -t tcp协议
      -uudp协议
      -l 监听
      -r路由
      -n显示ip地址和端口

  • mount

    1. scsi + rom:small computer system interface(小型计算机系统接口),sr0:rom接口上的第一个文件
    2. /dev/cdrom 一个软链接
      ls -l /dev/cdrom
      lrwxrwxrwx. 1 root root 3 6月 16 03:33 /dev/cdrom -> sr0
    3. mount /dev/cdrom /mnt/cdrom(写成/dev/sr0一样)
    4. umount /dev/cdrom(取消挂载<–>拔出U盘) 因为已经连接上了umount /mnt/cdrom也行

关机等

  • shutdown 关机
    1. shutdown -h关机后停机
    2. shutdown -r 关机后重启
    3. halt
      poweroff
      init 0等
    4. reboot
  • init的配置文件/etc/inittab
    • 查看以及修改系统的runlevel
    1. runlevel查看系统运行级别
  • logout退出登录
